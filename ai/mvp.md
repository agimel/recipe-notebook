# Recipe notebook application - MVP document

## Goal
Create a simple recipe notebook app that helps to save various recipes. Each recipe should have title, ingredients section and steps sections. User can add category to the recipe, eg. dessert, soup, cake. 

## Core features in scope
1. Save recipe
2. Label recipe with a category ( 1 or more)
3. Add new category
4. Search by title
5. List all in a category

## Features not in scope
1. Search by ingredients

## Success criteria
1. User is able to add recipe
2. User is able to add category
3. User is able to see all recipes in selected category
