# Recipe notebook application - MVP document

## Goal
Create a simple recipe notebook web application that helps to save various recipes. Each recipe should have title, ingredients section,steps sections, difficulty level, cooking time. User can add category to the recipe, eg. dessert, soup, cake. Recipe can have more than one category.

## Core features in scope
1. Save/edit/delete recipe
2. Label recipe with a category ( 1 or more)
3. Add new category
4. Search by title
5. List all recipes in a category
6. Create an account

## Features not in scope
1. Search by ingredients
2. Delete categories
3. Share recipes
4. Import/export recipes

## Success criteria
1. User is able to add recipe
2. User is able to add category
3. User is able to see all recipes in selected category
